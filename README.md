# prueba-castores-olmos
El programa esta desarrollado en Spring Boot, javascript vanilla mayormente, y algunas librerias y CSS, 
la parte de datos esta en Mysql y se manejan las entidades con la suite de JPA el ide de desarrollo que utilice
fue Visual studio code
# Versiones
- Mysql server 8.4.6
- java (proyecto) <java.version>17</java.version>
- java 8 update 471
- Spring Boot 4.0.0<
- Visual Studio Code Version: 1.106.3
# Ejecucion
El archivo de configuraciones que esta ubicado en main/resources/aplication.properties tiene la configuracion
de la base de datos asi que habria que modificarlo, ademas esta configurado para crear las tablas cuando se ejecuta por primera vez
solo hay que llamar igual a la base de datos, tambien comparto el script de base de datos con el que hice las pruebas,
para ejecutar el programa simplemente hay que complilar y ejecutar la clase main "DemoApplication.java" esto levantara el tomcat 
por el puerto predeterminado 8080
